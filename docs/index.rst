casaconfig
==========

As part of the `Common Astronomy Software Applications (CASA) <https://casadocs.readthedocs.io/en/stable/>`_ suite, the casaconfig
package provides an interface to the configuration file(s) and external data sources necessary for a CASA data reduction session.
